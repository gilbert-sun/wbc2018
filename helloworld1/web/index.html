課程介紹

該課程會教大家學習 HTML5 以及 CSS3，先為製作網站打造一個良好的基礎。此外還會使用 Bootstrap 這套框架帶大家製作 RWD 網站，讓網站可以在手機 / 平板 / 桌機等裝置輕鬆閱讀瀏覽。

基礎課程會用到簡易的 jQuery 與 CSS3 動畫，教大家如何使用 jQuery plugin，讓網站更加活潑生動。

課程大綱

認識 HTML 標籤與語意化結構
CSS 選擇器
RWD 佈局介紹
Bootstrap 介紹與應用
